How to use?

Simple. In your C program, add this line: 

#include "cn_vec.h"
From there on out, you can now use the functions in the CN_Vec library. Make sure both "cn_vec.c" and "cn_vec.h" are in yout directory with your program.

List of functions

On this domain, I have included a manual for every function in the library. Feel free to check it out if you have any questions in regards to what functions the library contains.

Here is a list of every function available separated in categories:

Initializer

  • new_cn_vec (Use cn_vec_init instead)

Add

  • cn_vec_push_back
  • cn_vec_insert

Set

  • cn_vec_resize
  • cn_vec_set
  • cn_vec_delete
  • cn_vec_copy

Modify

  • cn_vec_clear
  • cn_vec_swap
  • cn_vec_reverse
